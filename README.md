# CodeVision-AI

CodeVision-AI is a web-based coding practice platform built using Python, Flask, HTML, CSS, JavaScript, and SQLite.

## Features
- User Login & Registration
- Browse Coding Problems
- Search Problems
- Mark Problems as Solved
- Favorite Problems
- Progress Tracker
- Leaderboard
- User Profile

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository.
2. Install Flask:
   pip install flask
3. Run:
   python app.py
4. Open http://127.0.0.1:5000 in your browser.
